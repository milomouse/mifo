mifo (2.5)
---------

About:

* MPlayer daemon with preset/native commands controlled through a FIFO.

* Written in ZSH.

---------

Requirements:

* ZSH

* MPLAYER

* PROCPS (ps|pgrep|pkill)

* COREUTILS (mkfifo)

*  Start daemon:

    % mifo --init

*  Quit daemon:

    % mifo --quit

* Show help:

    % mifo --help

* Show detailed help for an option:

    % mifo --OPTION --
